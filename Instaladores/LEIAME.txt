------+    WINDOWS    +-------

Descompacte a pasta windows e copie o arquivo gerador_image_de_versiculo.exe para Área de Trabalho ou Pendrive. O App foi feito para ser portátil. Basta um duplomclique e ele funcionará.
 
  
 
------+    UBUNTU    +-------

Abra o terminal com CTRL+Alt+T, e digite os comandos abaixo:

sudo apt install unzip
unzip ubuntu
sudo ./install

Agora pesquise por Gerador de Imagens de Versículos entre seus Aplicativos
